# Simple-Blog
A Simple blog application built with HTML, CSS, and JavaScript where users can create, edit, delete and search blog posts.The project demonstrates the implementation of CRUD (Create, Read, Update, Delete) operations without requiring a backend or database.

Blog posts are stored using the browser's Local Storage, ensuring that data remains available even after the page is refreshed. Each post includes a title, content, and the date and time of publication. The application also features a search function that enables users to quickly find posts by matching keywords in the title or content.

This project showcases core front-end development concepts such as DOM manipulation, event handling, responsive web design, form validation, and local data storage. It is designed as a beginner-friendly project that demonstrates practical JavaScript skills and serves as a strong portfolio project for internships and entry-level front-end development roles.
